# PSP - Prompt Suggestion Protocol

This project is primarily an RFC specification for defining a tagging architecture for Large Language Models (LLMs).

## Project Structure

This repository is organized into the following components:

### `/standard/`
Contains the RFC specification document defining the PSP tagging architecture. This is the primary focus of the project.

### `/implementations/`
Contains reference implementations of the PSP standard in various forms:

- **`/implementations/api/`** - API server implementations in different programming languages
- **`/implementations/mcp-servers/`** - Model Context Protocol (MCP) server implementations in different programming languages

## Getting Started

To review or contribute to the specification, see the [standard/](./standard/) directory.

For implementation examples, explore the [implementations/](./implementations/) directory.
