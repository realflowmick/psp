# PSP Standard Specification

This directory contains the official RFC specification for the Prompt Suggestion Protocol (PSP).

## Documents

- **[SPEC.md](./SPEC.md)** - The main PSP specification document defining the tagging architecture for Large Language Models

## About the Specification

The PSP specification defines:
- A standardized tagging architecture for LLM prompts
- Core tag types and namespaces
- Implementation requirements for PSP-compliant systems
- Guidelines for extending the protocol

## Contributing to the Specification

The specification is the primary focus of this project. Contributions, suggestions, and feedback are welcome:

1. Review the [SPEC.md](./SPEC.md) document
2. Open an issue to discuss proposed changes
3. Submit a pull request with your suggested edits
